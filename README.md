# CROP-RILAB

This repository contains the main project modules for CROP-RILAB.

To understand or run each part of the project, open the README inside the relevant folder:

- [Dashboard](Dashboard/README.md): React dashboard, MapServer setup, Docker configuration, and data extraction notes.
- [Crop Classification](CropClassisification/README.md): Crop classification workspace for scripts, notebooks, experiments, and model documentation.

## Repository Structure

```text
CROP-RILAB/
  Dashboard/
    README.md
  CropClassisification/
    README.md
```

## Getting Started

1. Open the folder for the part of the project you want to work on.
2. Read that folder's `README.md`.
3. Follow the setup and usage instructions written there.

## Notes

- The dashboard module is in `Dashboard/`.
- The crop classification folder is currently named `CropClassisification/` in the repository.
